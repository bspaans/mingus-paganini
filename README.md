The Paganini Arpeggiator
========================


# Install

```
python setup.py install
```


# Run

```
paganini input.nini output.mid
```

# The nini file format

```
bpm: 150
updown: Cmaj7 -octave, 64 x 1
updown: Fmaj7 -octave, 64 x 1
updown: Cmaj7 -octave, 64 x 1
updown: Gdom7 -octave +octave, 64 x 1

```
